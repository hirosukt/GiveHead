# GiveHead
The plugin add command to get player's head.  
Tested in: 1.12.2

Page : 

## Usage
/head {optional: player}  
/h {optional: player}
/kubi {optional: player}
/namakubi {optional: player}
/skull {optional: player}
/playerskull {optional: player}

to get player's head.
